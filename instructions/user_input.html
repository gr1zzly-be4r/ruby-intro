The original course did most of the "user input" from little dialog boxes that sprung up from Eclipse. I did not know how to do something similar in Ruby, so the user input in this repository is all command-line based.
Basic premise of this lab is to make a program that calculates the amount of calories that a food contains from user supplied input. Think "x food has 25 grams of fat, how many calories would this be?" Some general rules: ***Disclaimer: I have no idea whether these calculations are nutritionally accurate***
  • A gram of carbohydrates is 4 calories.
  • A gram of fat is 9 calories.
  • A gram of protein is 4 calories.
The program needs to do 3 things:
  1. Ask the user for a food.
  2. Get information about that food.
  3. Process the information about that food and give that information back to the user.
An example output could be:

Snickers has x amount of calories: x amount of grams of fat x amount of grams of carbohydrates x amount of grams of protein This means that you have x total calories. Should you eat this food? [yes/no determined by coin flip]
